# Supercar Performance Analysis Dashboard: Nürburgring & Engineering Comparison

A professional data visualization dashboard designed to evaluate supercar track capability and strategic market positioning through the lens of Nürburgring lap times and core engineering metrics.

## 🏎️ Overview

This project analyzes the performance of top-tier automotive brands (BMW, Ferrari, Lamborghini, Mercedes-AMG, and Porsche) by correlating technical specifications with real-world track performance at the iconic Nürburgring Nordschleife.

## 📊 Key Metrics Tracked

* **Best Power to Weight Ratio:** 0.63 (Leading efficiency metric)
* **Fastest 0-100 Km/h:** 2.80 seconds (Acceleration benchmark)
* **Fastest Lap Time:** 395.18 seconds (6:35.18)
* **Top Speed at Nürburgring:** 352.00 km/h

## 📈 Visualizations Included

1.  **Nürburgring Lap Time by Model:** A trend analysis across specific models including the Ferrari 296 GTB, 488 Pista, Porsche 911 GT2 RS MR, AMG One, and Lamborghini Aventador SVJ.
2.  **Cars Under 7 Min by Brand:** A comparative bar chart showing which manufacturers have the highest volume of sub-7-minute production cars (Lamborghini, Mercedes-AMG, and Porsche leading).
3.  **Performance vs. Engineering:** A scatter plot mapping technical engineering specs against track performance to identify "sweet spots" in automotive design.
4.  **Segment/Class Performance:** A waterfall/step chart showing lap time variations between 'Track', 'Supercar', and 'Hypercar' classifications.

## 🛠️ Tech Stack & Tools

* **Data Source:** Excel
* **Visualization Tool:** Power BI 
* **Design Theme:** High-contrast performance aesthetic (Racing Red & White).

## 🚀 Key Insights

* **Brand Dominance:** Porsche and Mercedes-AMG show consistent performance leadership in the sub-7-minute category.
* **The "Hypercar" Gap:** Data reveals the significant performance jump between standard supercars and limited-run hypercars like the AMG One.
* **Weight Matters:** The dashboard highlights the strong correlation between high power-to-weight ratios and lower lap times over raw horsepower.
